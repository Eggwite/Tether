| Method | Path                    | Description                      |
|--------|-------------------------|----------------------------------|
| `GET`    | `/v1/users/{userID}`    | Get presence snapshot for a user |
| `GET`    | `/healthz`              | Health/readiness probe           |
| `WS`     | `/socket`               | WebSocket gateway for presence   |
