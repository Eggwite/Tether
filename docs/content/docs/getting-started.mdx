## Quickstart: HTTP

```bash
curl http://localhost:8080/v1/users/1447110828783566973
```

## Quickstart: WebSocket

```js
const ws = new WebSocket('ws://localhost:8080/socket');
ws.onmessage = (evt) => {
  const msg = JSON.parse(evt.data);
  if (msg.op === 1) {
    // hello
    ws.send(JSON.stringify({ op: 2, d: { subscribe_to_ids: ['1447110828783566973'] } }));
  } else if (msg.op === 0 && msg.t === 'PRESENCE_UPDATE') {
    console.log('presence', msg.d);
  }
};
```

> **Tip:** See the [Presence Schema](./presence) for the data shape returned.
