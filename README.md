
# Tether

Tether streams your Discord presence and activities to your website via REST APIs and WebSocket connections, powered by a lightweight Go service that deploys in seconds. Built for developers, it automatically handles Discord IDs, hashes, and URLs. No manual translation needed. Kindly consider giving the repository a â­ star.

## Quickstart


Visit here to visit the quickstart on the docs. [Documentation](https://tether-ebon-kappa.vercel.app/docs)

Join the Tether Discord server [here](https://discord.gg/ZggGDhjCyE) to start observing your presence. (discord.gg/ZggGDhjCyE)

> [!IMPORTANT]  
> For those not self-hosting, you must join the discord server to let the API observe your presence.
